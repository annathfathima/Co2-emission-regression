# CO2 Emission Prediction using Machine Learning

## Project Overview
This project predicts CO2 emissions using a Machine Learning Regression model. The project includes data preprocessing, model training, evaluation, and prediction.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- XGBoost

## Files
- CO2_emission.csv – Dataset
- emission2.ipynb – Jupyter Notebook containing the complete project

## Project Workflow
1. Load the dataset
2. Data preprocessing
3. Train the regression model
4. Evaluate model performance
5. Predict CO2 emissions
